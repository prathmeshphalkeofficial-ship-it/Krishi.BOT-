## 🤖 API Routes Summary

### `/api/chat` (POST)
- Input: `{ prompt: string, language?: string }`
- Output: `{ text: string }`
- Model: `llama-3.3-70b-versatile`
- Used by: Soil page chat, News/Mandi chat, general chat

### `/api/chat-vision` (POST)
- Input: `{ message: string, imageBase64: string, language?: string, context?: string }`
- Output: `{ text: string }`
- Model: `llama-4-scout-17b-16e-instruct` (vision) when image present, falls back to text model
- Used by: Disease page chat popup
- Key feature: Sends actual image to vision model so AI can SEE the plant

### `/api/disease` (POST)
- Input: FormData with `image` file
- Output: Full disease analysis JSON
- Model: `llama-4-scout-17b-16e-instruct`

### `/api/voice` (POST)
- Input: `{ prompt, language, lat?, lon? }`
- Output: `{ text: string }`
- maxDuration: 30 seconds
- Features: Motor command detection → intent detection → live data fetch → Groq answer
- Intent keywords: weather/forecast/news/mandi + all commodity names in en/hi/mr

### `/api/mandi` (GET/POST)
- Input: `{ district: string }`
- Output: `{ success, records[], source, sourceUrl }`
- Scrapes vegetablemarketprice.com, falls back to mock data
